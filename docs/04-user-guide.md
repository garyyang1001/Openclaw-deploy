# OpenClaw AI 助理 — 使用指南

> 這是一份給一般使用者的說明，幫助你了解 AI 助理有哪些功能，以及如何使用。

---

## 什麼是 OpenClaw？

OpenClaw 是你的**個人 AI 助理**，可以透過你常用的通訊軟體（Telegram、WhatsApp、Discord 等）與你對話。它不只是聊天機器人——它能幫你搜尋資料、瀏覽網頁、排程任務，甚至執行程式。

---

## 如何開始使用

### 方法一：透過 Telegram

1. 在 Telegram 搜尋你的 Bot（部署時提供的 Bot 名稱）
2. 點擊「開始」或傳送任何訊息
3. 系統會給你一個**配對碼**（6 位數字）
4. 將配對碼提供給管理員核准
5. 核准後即可開始對話

### 方法二：透過網頁介面

1. 開啟瀏覽器，前往你的 OpenClaw 網址（例如 `https://你的域名.zeabur.app`）
2. 這是 **OpenClaw Control UI**（管理控制台）
3. 登入後可以在 WebChat 中直接對話

## 基本功能

### 日常對話

直接傳訊息給 Bot，它會用 AI 回答你：

```
你：今天台北天氣如何？
Bot：[AI 回答]

你：幫我翻譯這段英文：Hello, how are you?
Bot：你好，你好嗎？
```

### 常用指令

在對話中輸入以下指令：

| 指令 | 功能 |
|------|------|
| `/status` | 查看 Bot 目前狀態 |
| `/reset` | 重置對話（清除上下文） |
| `/model` | 查看或切換 AI 模型 |
| `/model list` | 列出所有可用的 AI 模型 |
| `/help` | 顯示幫助資訊 |

### 切換 AI 模型

如果你有多個 AI 模型可用，可以隨時切換：

```
/model list
→ 1. kimi-coding/k2p5 (目前使用)
→ 2. anthropic/claude-opus-4-5
→ 3. openai/gpt-4o

/model 2
→ 已切換至 anthropic/claude-opus-4-5
```

## 進階功能

### 網路搜尋

AI 可以幫你搜尋最新資訊（需設定 Brave Search API）：

```
你：幫我查一下 2026 年台灣的最低工資是多少
Bot：[搜尋結果 + AI 整理的答案]
```

### 圖片分析

傳送圖片給 Bot，它可以：
- 描述圖片內容
- 分析圖表數據
- 辨識文字（OCR）
- 回答關於圖片的問題

```
[傳送一張菜單照片]
你：這個菜單上最貴的菜是什麼？
Bot：根據菜單，最貴的是...
```

### 長文件處理

AI 支援超長上下文（最高 256K token），你可以：
- 貼上長篇文章請它摘要
- 傳送文件請它分析
- 持續多輪對話，AI 會記住前面的內容

### 排程任務

請 AI 設定定時提醒或任務：

```
你：每天早上 9 點提醒我喝水
Bot：已設定每日 09:00 提醒
```

## 群組使用

### 邀請 Bot 進群組

1. 將 Bot 加入 Telegram/Discord 群組
2. 在群組中使用 `@BotName` 提及 Bot 才會回應
3. 管理員可設定為「所有訊息都回應」模式

### 群組指令

```
@BotName 幫我摘要今天的討論重點
@BotName 翻譯上面那段話成英文
```

## 安全與隱私

### 你的資料是安全的

- **配對制度**：只有經過管理員核准的人才能使用 Bot
- **對話隔離**：每個人的對話是獨立的，互不影響
- **資料位置**：所有資料存在你自己的伺服器上
- **加密傳輸**：所有通訊透過 HTTPS 加密

### 安全提醒

- 不要在對話中分享密碼、信用卡號等敏感資料
- AI 會自動遮蔽偵測到的敏感資訊
- 如有疑慮，使用 `/reset` 清除對話記錄

## 常見問題

### Bot 沒有回應？

1. 確認 Bot 是否在線（傳送 `/status`）
2. 如果在群組中，確認是否有 `@` 提及 Bot
3. 聯繫管理員確認服務是否正常運作

### 回應速度很慢？

- AI 處理複雜問題需要時間（通常 5-30 秒）
- 如果使用思考模式，可能需要更長時間
- 嘗試切換到更快的模型（`/model list`）

### 回答不準確？

- 使用 `/reset` 清除上下文，重新開始
- 嘗試更具體地描述你的問題
- 切換到更強的模型（如 Claude Opus）

### 如何查看使用量？

- 在 Control UI 中可以查看使用統計
- AI API 費用由你的 API Key 對應的帳號承擔

## 管理員功能

以下功能僅限管理員使用：

### Control UI（控制台）

前往 `https://你的域名.zeabur.app` 可以：
- 查看所有對話
- 管理配對請求
- 調整設定
- 查看系統日誌

### 配對管理

```bash
# 列出待核准的配對請求
openclaw pairing list telegram

# 核准配對
openclaw pairing approve telegram <CODE>
```

### 安全稽核

```bash
openclaw security audit --deep
```
